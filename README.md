# Portfolio Website

Personal portfolio website built with Next.js, showcasing the projects that I have worked on and my background as a full-stack developer.

## Pages

- **Home** - Introduction and links to projects and contact
- **Projects** - Overview of personal and academic projects
- **About** - Skills, background, and experience

## Tech Stack

- [Next.js 15](https://nextjs.org) (App Router)
- [Tailwind CSS v4](https://tailwindcss.com)
- [next-themes](https://github.com/pacocoursey/next-themes) - system, light, and dark mode support
- [Geist](https://vercel.com/font) font

## Running Locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
